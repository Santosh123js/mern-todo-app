# MERN ToDo App

## Features

- User Signup and Login
- JWT Authentication
- Add New Tasks
- Update Tasks
- Delete Tasks
- View All Tasks (List)
- Responsive User Interface

## Tech Stack

### Frontend

- React.js
- CSS

### Backend

- Node.js
- Express.js

### Database

- MongoDB

## Installation

### Frontend

cd Frontend

npm install

npm run dev

### Backend

cd backend

npm install

npm start
